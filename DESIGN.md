# Design System: Prisma

## 1. Overview

**Creative North Star: "The White Studio"**

Prisma One is a precision machine photographed under perfect light. The system is a
gallery-clean **optical white** stage where a single physical object — a silver
carbon aero bicycle — is presented like an instrument on a studio floor. The bike
is the hero; the page is the vitrine. Color is not painted onto the page: the bike
is near-monochrome silver-and-black, and the only chromatic note is a restrained
**spectrum accent** (the brand's namesake light), used as a thread, never a wash.
Everything else is white space, exact alignment, and confident type. The mood is
Apple product-launch precision crossed with Teenage Engineering's industrial restraint.

This system explicitly rejects the **corporate / stock** look named in PRODUCT.md —
no corporate-blue palettes, no stock photography, no templated enterprise sections.
It equally rejects the 2026 AI defaults: no warm cream/sand "paper" background
masquerading as elegance (the surface is a *true* optical white at chroma 0, not a
warm-tinted near-white), no gradient text, no decorative glassmorphism, no
literal rainbow. The spectrum here is optical and earned — it appears only where
refraction would actually put it.

**Key Characteristics:**
- True optical-white surface (neutral, chroma 0) — gallery vitrine, not warm paper
- Refracted spectrum as the *only* color, produced by the object, used sparingly
- Bold sans display + clean neutral body — editorial confidence, modern precision
- Choreographed motion: light moves, dispersion animates, the object reveals on scroll
- Extreme restraint: vast white space, exact grid, every element deliberate

## 2. Colors

A near-monochrome optical-white system in which the full visible spectrum appears
only as refracted light — disciplined, physical, never decorative.

### Primary
- **Optical White** `[to be resolved during implementation — true neutral, ~oklch(98% 0 0), chroma 0]`: The surface. The vitrine the object sits in. Not cream, not sand, not warm paper — a clean, near-pure neutral white that reads as gallery light.

### Secondary
- **Refracted Spectrum** `[to be resolved — a violet→indigo→cyan→green→amber→red dispersion, oklch high-chroma stops]`: The signature. Generated by the prism object: caustics, edge fringing, a thrown spectral band. Appears only where light passes through glass. Never a flat background gradient.

### Neutral
- **Graphite Ink** `[to be resolved — near-black, ~oklch(20% 0 0)]`: Display and body text on white. Must hit ≥4.5:1 (trivially clears it against optical white).
- **Mid Slate** `[to be resolved — ~oklch(50% 0 0)]`: Secondary text, captions, technical labels. Verify ≥4.5:1 for body sizes.
- **Hairline Grey** `[to be resolved — ~oklch(92% 0 0)]`: 1px dividers, spec-table rules, input strokes.

### Named Rules
**The Spectrum-As-Thread Rule.** Spectrum color is a restrained brand accent — a
hairline, a single divider, a small mark — never a wash, never gradient text, never
a literal rainbow band across a section. The page is near-monochrome (white + the
bike's silver/black); the spectrum is the one quiet signature that earns the name.

**The Chroma-Zero Surface Rule.** The body background is a true neutral white at
chroma 0. No warm tint "for elegance." If it reads as cream, sand, paper, or ivory,
it is wrong — that is the AI default, and it kills the cold-optical-lab feel.

## 3. Typography

**Display Font:** `[bold geometric/grotesque sans — to be chosen at implementation]`
**Body Font:** `[clean neutral sans — to be chosen at implementation]`
**Label/Mono Font:** `[a monospace for spec readouts — optional, to be chosen]`

**Character:** Big, confident, modern sans display paired with a quiet neutral body —
editorial impact without ornament. Pair on a contrast axis (weight/scale), never two
near-identical grotesques. The display states; the body explains.

### Hierarchy
- **Display** (bold, `clamp()` max ≤ 6rem / ~96px, line-height ~1, letter-spacing ≥ -0.04em): Hero product name and section statements. Use `text-wrap: balance`.
- **Headline** (semibold, ~2-3rem): Section leads.
- **Title** (medium, ~1.25-1.5rem): Sub-section and feature names.
- **Body** (regular, ~1-1.125rem, line-height ~1.6, max 65–75ch): Prose. `text-wrap: pretty`.
- **Label** (medium, ~0.75-0.875rem, optionally mono, slight tracking): Technical spec readouts, measurements, captions.

### Named Rules
**The One-Voice-Loud Rule.** Exactly one display moment per viewport. The page is
quiet so the one bold statement lands. Two competing display headlines on one screen
is forbidden.

## 4. Elevation

Choreographed motion, but a near-flat material world. Depth comes from the *object's*
own real shadow and caustics on the white surface — not from UI drop-shadows on cards.
Surfaces are flat at rest; the only true shadow on the page is the one the glass prism
casts. Avoid generic ambient card shadows; they read as templated SaaS.

### Named Rules
**The Only-The-Object-Casts-Shadow Rule.** UI elements (text blocks, dividers, the
nav) are flat. The single real shadow belongs to the prism. Everything else stays on
the white plane.

## 6. Do's and Don'ts

### Do:
- **Do** keep the surface a true optical white at chroma 0 (~oklch(98% 0 0)).
- **Do** produce spectrum color only as refraction from the glass object — caustics, edge fringing, thrown spectral bands.
- **Do** lean on vast white space and exact alignment; restraint is the premium signal.
- **Do** make motion choreographed and purposeful (scroll-driven reveal, animated dispersion) — and ship a `prefers-reduced-motion: reduce` crossfade/static fallback for every one.
- **Do** verify body text ≥4.5:1 and large text ≥3:1 against the white surface.
- **Do** let the object cast the page's one real shadow.

### Don't:
- **Don't** use a warm cream/sand/paper/ivory background — that's the saturated AI default and the opposite of an optics lab. Chroma 0, not chroma-warm.
- **Don't** use corporate-blue palettes, stock photography, or templated enterprise sections (PRODUCT.md anti-reference: *corporate / stock*).
- **Don't** paint a flat rainbow gradient as a section background, or use `background-clip: text` gradient text. Spectrum is refracted, never applied.
- **Don't** add decorative glassmorphism, ambient card drop-shadows, hero-metric blocks, identical card grids, or an eyebrow kicker above every section.
- **Don't** run two display-scale headlines in one viewport — one loud voice at a time.
